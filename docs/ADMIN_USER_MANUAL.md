This guide is designed for **Administrators** managing the BotsJobsConnect platform. Learn how to manage users, approve companies, oversee jobs, and maintain platform security.

### Admin Account

## User Management

### Viewing Users

1. Go to **Admin Panel ‚Üí User Management**
2. **Filter by:**
   - User Type (Job Seeker, Employer, Trainer)
   - Account Status (Active, Suspended, Pending)
   - Verification Status
   - Registration Date

### User Actions

**For Each User, You Can:**

- **View Profile** - See complete user information
- **Edit Profile** - Update user details (if needed)
- **Suspend Account** - Temporarily disable account
- **Delete Account** - Permanently remove account
- **Change Role** - Modify user role/permissions
- **View Activity** - See user's platform activity

### Granting Admin Access

To make a user an administrator:

1. Go to **User Management**
2. Select the user
3. Tap **"Grant Admin Access"**
4. Confirm the action
5. User will receive notification

**Note:** Only existing admins can grant admin access.

### User Statistics

View platform-wide statistics:
- Total registered users
- Active users (last 30 days)
- New registrations (this month)
- User type distribution
- Verification rates

## Company Verification (KYC)

### Reviewing Company Submissions

1. Go to **Admin Panel ‚Üí Company Approvals**
2. See all pending verifications
3. **For each company, view:**
   - Company name and details
   - Submitted documents
   - Submission date
   - Previous rejection reasons (if any)

### Required Documents Checklist

Verify these documents are present:

- ‚úÖ **CIPA Certificate** - Valid incorporation certificate
- ‚úÖ **CIPA Extract** - Recent extract (within 6 months)
- ‚úÖ **BURS TIN Evidence** - Tax registration proof
- ‚úÖ **Proof of Business Address** - Valid address document

### Approving a Company

1. **Review Documents:**
   - Check document validity
   - Verify information matches
   - Ensure all required docs present

2. **Approve:**
   - Tap **"Approve"** button
   - Add approval notes (optional)
   - Confirm approval
   - Company receives notification

3. **Result:**
   - Company status changes to "Approved"
   - Company can now post jobs
   - Notification sent to company

### Rejecting a Company

1. **Review Issues:**
   - Identify missing/invalid documents
   - Note discrepancies
   - Document rejection reason

2. **Reject:**
   - Tap **"Reject"** button
   - **Enter Rejection Reason** (required):
     - Missing documents
     - Invalid documents
     - Information mismatch
     - Other (specify)
   - Confirm rejection
   - Company receives notification with reason

3. **Result:**
   - Company status changes to "Rejected"
   - Company can resubmit after fixing issues
   - Rejection reason visible to company

### Verification Best Practices

1. **Review Thoroughly:**
   - Check all documents carefully
   - Verify document authenticity
   - Ensure information consistency

2. **Respond Promptly:**
   - Review within 1-3 business days
   - Don't leave companies waiting
   - Provide clear feedback

3. **Be Consistent:**
   - Apply same standards to all companies
   - Document rejection reasons clearly
   - Follow verification guidelines

4. **Communicate:**
   - Provide helpful rejection reasons
   - Guide companies on corrections
   - Answer verification questions

## Notification Management

### Sending Platform Notifications

1. Go to **Admin Panel ‚Üí Notifications**
2. Tap **"Send Notification"**
3. **Select Recipients:**
   - All Users
   - Job Seekers Only
   - Employers Only
   - Trainers Only

4. **Compose Message:**
   - Title (required)
   - Message body
   - Action button (optional)
   - Link/URL (optional)

5. **Schedule:**
   - Send immediately
   - Schedule for later
   - Set expiration date

## Troubleshooting

### Cannot Access Admin Panel

**Possible Issues:**
- Account not granted admin access
- Incorrect credentials
- App cache issues

**Solutions:**
- Verify admin status in database
- Check email/password
- Clear app cache
- Contact development team

### Company Documents Not Loading

**Possible Issues:**
- Firebase Storage issues
- Large file sizes
- Network problems

**Solutions:**
- Check Firebase Storage status
- Wait for large files to load
- Check internet connection
- Try refreshing

### Notifications Not Sending

**Possible Issues:**
- FCM configuration issues
- Invalid user tokens
- Cloud Functions errors

**Solutions:**
- Check Firebase Cloud Messaging
- Verify FCM tokens
- Review Cloud Functions logs
- Test with small group first

### User Management Errors

**Possible Issues:**
- Firestore permission errors
- Database connection issues
- Invalid user data

**Solutions:**
- Check Firestore rules
- Verify database connection
- Review user data structure
- Contact development team
