Troubleshooting

Cannot Access Admin Panel

Possible Issues: - Account not granted admin access - Incorrect credentials - App cache issues

Solutions: - Verify admin status in database - Check email/password - Clear app cache - Contact development team

Company Documents Not Loading

Possible Issues: - Firebase Storage issues - Large file sizes - Network problems

Solutions: - Check Firebase Storage status - Wait for large files to load - Check internet connection - Try refreshing

Notifications Not Sending

Possible Issues: - FCM configuration issues - Invalid user tokens - Cloud Functions errors

Solutions: - Check Firebase Cloud Messaging - Verify FCM tokens - Review Cloud Functions logs - Test with small group first

User Management Errors

Possible Issues: - Firestore permission errors - Database connection issues - Invalid user data

Solutions: - Check Firestore rules - Verify database connection - Review user data structure - Contact development team
